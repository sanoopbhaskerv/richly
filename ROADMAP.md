# SB Editor roadmap

## Current release

The current package includes a dependency-free editor core, React bindings,
formatting, lists, links, images, source editing, fullscreen mode, and an
advanced table experience with resize handles and right-click actions.

## Next priorities

1. Improve input method editor (IME), paste, and selection edge-case coverage.
2. Add multi-cell selection plus table merge/split operations.
3. Add search and replace, word count options, preview, and visual block tools.
4. Add image upload hooks and image resize handles.
5. Add a framework-neutral plugin authoring guide and extension examples.

## Before a stable 1.0

- Establish a semantic-versioning and release process.
- Publish an accessibility support statement.
- Add a public changelog and migration policy.
- Expand browser coverage with real-world paste fixtures and IME scenarios.
