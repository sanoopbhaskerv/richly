# Richly roadmap

## Current release

The current package includes a dependency-free editor core, React bindings,
formatting, lists, links, images, source editing, fullscreen mode, and an
advanced table experience with multi-cell merge/split, resize handles,
right-click actions, and an inline options toolbar that floats beneath the
selected table (`table-inline-toolbar`, actions `inline-table-action-<id>`).
It also includes IME-aware editing, safe paste handling, search and replace,
configurable counts, preview, and visual blocks.

## Next priorities

1. Add image upload hooks and image resize handles.
2. Add a framework-neutral plugin authoring guide and extension examples.
3. Expand find/replace with match counts and keyboard navigation.

## Before a stable 1.0

- Complete manual assistive-technology testing for the published support
  matrix.
- Add keyboard controls and live-region feedback for table range selection and
  pointer-based resize handles.
- Grow the real-world paste corpus as browser and office-suite output evolves.
