# SB Editor roadmap

## Current release

The current package includes a dependency-free editor core, React bindings,
formatting, lists, links, images, source editing, fullscreen mode, and an
advanced table experience with multi-cell merge/split, resize handles, and
right-click actions. It also includes IME-aware editing, safe paste handling,
search and replace, configurable counts, preview, and visual blocks.

## Next priorities

1. Add image upload hooks and image resize handles.
2. Add a framework-neutral plugin authoring guide and extension examples.
3. Expand find/replace with match counts and keyboard navigation.

## Before a stable 1.0

- Establish a semantic-versioning and release process.
- Publish an accessibility support statement.
- Add a public changelog and migration policy.
- Expand the paste fixture library with more Word and Google Docs samples.
